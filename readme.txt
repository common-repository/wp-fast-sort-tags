== Description ==

"Wp-Fast-Sort Tags" allows you to create a page with the list of all tags sorted by name and then have all the posts related to this tag.


== Installation ==

1. Copy the the whole folder "Wp-Fast-Sort-Tag" into the wp-content/plugins folder of your WordPress installation. 
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin through the 'Settings' menu in WordPress.
